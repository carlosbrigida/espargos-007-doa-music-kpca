# Organização dos Experimentos

## baseline
Scripts usados para reproduzir o uRoot-MUSIC original, com e sem CRAP.

## pca_csi
Experimentos com PCA aplicada diretamente ao CSI.

## pca_covariance
Experimentos com PCA aplicada à matriz de covariância R.

## kpca_covariance
Experimentos com KPCA aplicada à matriz de covariância R.

## physics_validation
Testes usados para verificar se a PCA(R) preserva a física do problema:
hermitianidade, positividade, autovalores e subespaço de ruído.

## cache_generation
Scripts de geração de cache, CRAP e splits.

## tables_reports
Scripts para gerar tabelas, relatórios e material para tese/BEEN.
